本服务提供的应用程序编程接口，用户通过 HTTP 请求获取随机图片（302 重定向至图片文件）或图片元数据（JSON 格式）。

**2.5 投稿者**

开发者对投稿作品进行审查的流程，审查标准包括但不限于：与米哈游游戏的相关性、图片清晰度、水印情况、是否含有不当内容等。审核结果包括但不限于：批准入库、驳回、要求修改等。

**3.2 服务功能**

**3.2.1 随机图片接口 (`/v1/random`)** — 返回一个随机图片资源。支持以下查询参数：

- `json=1`：返回 JSON 格式的图片元数据（文件 URL、游戏、尺寸、色调、画师等），而非 302 重定向；
- `game=genshin|hsr|zzz`：限定返回指定游戏的图片；
- `orientation=landscape|portrait|square`：限定返回指定方向的图片。

**3.2.2 分类统计接口 (`/v1/categories`)** — 返回各游戏的图片收录数量统计。

**3.2.3 网站页面** — 本服务提供首页文档、投稿指南、用户协议及隐私政策等静态信息页面。
