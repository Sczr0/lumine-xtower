本服务收录的图片数量较大，审核难免有遗漏。如果您发现以下问题，欢迎反馈。

## 可以反馈什么

| 问题类型 | 说明 | 举例 |
|---------|------|------|
| **AI 生成未标注** | 明显是 AI 生成但未标记 | 手指异常、不合理光影、涂抹感 |
| **标签/角色错误** | 角色名标错或遗漏 | 图上是荧却标了空 |
| **内容违规** | 包含不当内容 | 涉政、血腥、R18 |
| **版权问题** | 因版权问题希望移除图片 | 提供版权证明/作者公告 |
| **画师信息纠错** | 画师名写错或来源链接失效 | — |
| **图片画质问题** | 严重压缩失真、水印过大 | — |
| **其他问题** | 以上未列出的其他情况 | — |

## 反馈方式

请通过以下邮箱发送反馈，**务必在邮件中包含图片标识 (slug)**，否则无法定位：

- **电子邮箱**：xtower@xtower.site
- **邮件标题**：`[Lumine反馈] <问题类型>`
- **邮件内容**：
  1. 图片 slug（在 API JSON 响应的 `id` 字段中，或打标台显示的 `slug`）
  2. 问题类型（见上表）
  3. 简要说明

## 处理时效

提交后 3–5 个工作日处理。版权投诉按《用户协议》第 5.4 条的侵权处理机制执行。
