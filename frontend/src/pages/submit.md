> ### 投稿须知
>

>
> 1. 您确认拥有所投作品的完整版权，或已获得原作者的明确授权。严禁盗图或未经许可转载他人作品。

> 3. 您理解并接受，作品将作为 Lumine 随机图库的一部分，通过技术接口（API）开放给第三方用户调用。
> 4. Lumine 对投稿作品不主张任何所有权。您保留对自己作品的完整权利，并可随时通过反馈渠道请求撤回 Lumine 的分发许可。我们在收到请求并核实身份后，将在 14 个工作日内下架相关作品，但投稿者需理解并同意，您申请撤回已收录的作品时，在处理完成前的分发行为不受影响，该行为并非侵权行为。
> 5. 投稿作品须符合米哈游（HoYoverse）相关同人创作指引，且不得包含政治敏感、色情、血腥暴力或侵犯他人隐私的内容。
> 6. 本服务仅提供技术分发与展示平台，不对投稿内容的版权真实性承担任何法律连带责任。若收到相关权利人的版权主张，我们将遵循“先下架后核实”的原则处理。
>
> *特别的，若您请求 Lumine 收录**非本人作品**，除须遵守本站规则外， Lumine 的收录行为亦受到原作者版权声明的约束（如署名要求、来源标注等）。若原作者明确在图中声明“禁止转载”或“严禁二传”，Lumine 将拒绝收录该作品以尊重创作者意愿。由于人工审核的局限性，声明“禁止转载”或“严禁二传”的图片，我们也可能会错误收录入库，您可通过反馈渠道进行投诉。原作者特别声明或授权的，遵循其约定。（如原作者允许非商业转载/允许本服务使用等）*

## 投稿要求

| 项目 | 说明 |
|------|------|
| **格式** | PNG / JPG / WebP |
| **尺寸** | 建议不小于 1920×1080 |
| **水印** | 原则上不接受大面积水印或遮挡画面的文字 |
| **内容** | 米哈游旗下游戏相关（原神 / 崩坏：星穹铁道 / 绝区零 / 崩坏3） |
| **画风** | 不限，官方风格、同人均可 |

> 审核周期一般为 1–3 个工作日。驳回后会通过您留下的联系方式告知原因。

## 审核标准

- 内容必须与米哈游游戏相关
- 图片清晰度良好，无明显压缩失真
- 无水印或水印不影响观赏
- 不包含不当内容（血腥、政治敏感等）

## 常见驳回原因

- 图片含有「禁止转载」「请勿二次上传」等限制性水印
- 分辨率过低（单边小于 720px）
- 与米哈游游戏完全无关
- 内容重复（同一画师的同一张图已入库）
- 画面包含违规内容
